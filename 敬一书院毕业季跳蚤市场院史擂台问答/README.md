# 敬一书院院史擂台问答区

静态网页项目，题库共 90 道选择题。进入网页后随机抽取 10 道题进行作答，正确率 80% 及以上显示“可兑换一个印章盖至集章卡上”。

## 文件结构

- `index.html`：页面结构
- `styles.css`：暖橙校园市集风格样式
- `questions.js`：完整题库数据
- `app.js`：随机抽题、判分与答案展示逻辑
- `netlify.toml`：Netlify 静态部署配置

## 本地预览

可直接用浏览器打开 `index.html`，或在目录内启动任意静态服务器。

## Netlify

本项目无需构建命令，发布目录为当前根目录。
